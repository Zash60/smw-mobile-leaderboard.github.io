# SMW Speedrun Mobile Leaderboard

Leaderboard de speedrun para Super Mario World Mobile.

## Tecnologias

- React 18
- Vite
- Tailwind CSS v4
- Firebase (Auth + Realtime Database)
- Speedrun.com API

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

O conteúdo da pasta `dist/` deve ser servido como site estático.
